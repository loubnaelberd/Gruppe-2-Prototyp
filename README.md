# Global Finance Solutions – Banking Dashboard (Prototyp)

Banking-Prototyp im Stil moderner Apps (Revolut / N26 / Stripe). Die Anwendung dient ausschließlich als Frontend-Demo und arbeitet mit simulierten Daten – es werden keine echten Bankkonten oder Finanzdaten verarbeitet.

## Funktionen

- **Übersicht / Dashboard** – Alle wichtigen Finanzinformationen auf einen Blick: animierte Kennzahlen, eine Übersicht der wöchentlichen Ausgaben, interaktive Kartenansichten, Schnellzugriffe auf häufig genutzte Funktionen sowie eine einfache Budgetübersicht.
- **Transaktionen** – Eine übersichtliche Darstellung aller Buchungen mit Kategorien, Suchfunktion und praktischen Filtern, damit sich einzelne Vorgänge schnell finden lassen.
- **Kartenverwaltung** – Bankkarten können angezeigt und verwaltet werden. Funktionen wie Karten sperren oder entsperren sowie das Anpassen von Limits sind direkt integriert.
- **Digitaler Kontowechsel** – Der Kontowechsel wird in einer einfachen 3-Schritt-Journey simuliert: Identifikation, Bestätigung und Freigabe. Ergänzt wird der Prozess durch eine IBAN-Prüfung, eine PSD2-Simulation und die automatische Erkennung von Partnerbanken.
- **Mitarbeiteransicht / Vorgangsmanagement** – ür interne Teams steht eine Operations-Oberfläche zur Verfügung. Hier können Vorgänge gefiltert, durchsucht und bearbeitet werden. Mitarbeitende erhalten Einblick in Dokumente, Prüfungen und die Bearbeitungshistorie und können Fälle freigeben, zurückstellen oder zur manuellen Prüfung weiterleiten.
- **Sicherheit** – Der Prototyp zeigt typische Sicherheitsfunktionen moderner Banking-Anwendungen, darunter den Status der Zwei-Faktor-Authentifizierung, den Login-Verlauf sowie eine Einschätzung der Passwortstärke.
- **Benutzererlebnis** Die Anwendung unterstützt sowohl einen Dark Mode als auch ein responsives Design. Dadurch passt sich die Oberfläche automatisch an Desktop- und Mobilgeräte an und bietet auf Smartphones eine intuitive Navigation.
